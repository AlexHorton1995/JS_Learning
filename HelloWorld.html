Lesson 1 (taken from site Tutorial Republic): HTML Elements

What is an HTML Element? An HTML element is the individual component of an HTML document. It represents semantics. For example, the <a href> tag is a tag that allows a link to a different webpage. The tag will dictate the behavior of the html document in that section.
