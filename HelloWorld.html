Lesson 2 (taken from site Tutorial Republic) - HTML Elements

Html pages are made up of HTML Tags and Elements. What is an HTML Element? An HTML element is the individual component of an HTML document. It represents semantics. HTML tags are made up of HTML Elements.
For example, the <a href> tag is a tag that allows a link to a different webpage. The tag will dictate the behavior of the html document in that section.
